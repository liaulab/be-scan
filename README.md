# BE_SCAN
Base-Editing Screening Computational ANalysis (BE-SCAN) \
Authors: SPS, XYH

Extensive documentation for all modules/functions/parameters can be found here \
https://be-scan.readthedocs.io/en/main/index.html

Description: We are building a package for library design, quality control, analysis, and plotting for base editing and mutagenesis CRISPR screens. 

# Instructions for activating documentation locally

Run the following command from home directory be_scan to build the package
```
sphinx-autobuild docs/source docs/build/html/
```

or install BE-SCAN from terminal or in a notebook by running the following command
```
pip install git+https://github.com/liaulab/be-scan.git
```

Installation should take approximately 5 minutes.
